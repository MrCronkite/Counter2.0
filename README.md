# Counter2.0 🏋️‍♂️

**Counter2.0** — это iOS-приложение для подсчёта тренировок и подходов в спортзале.

## 💪 О проекте

Приложение помогает отслеживать количество выполненных упражнений и тренировочных сессий, делая процесс тренировок более структурированным и удобным.

## 🧠 Архитектура

В проекте практикуется **Clean Architecture**, что обеспечивает:
- разделение бизнес-логики и UI
- гибкость и масштабируемость приложения
- удобство тестирования и поддержки кода

## 🎯 Цель проекта

Основная цель — улучшение навыков архитектурного проектирования iOS-приложений и применение принципов чистого кода в реальном проекте.
